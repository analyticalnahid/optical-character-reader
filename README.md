# Image-To-Text-Converter
An image to text converter using deep learning library.

## How it works?
Checkout my youtube video-> https://youtu.be/tks_2EUGVLA

## Try Demo!
No need to install try it out -> https://analyticalnahid-image-to-text-converter-ocr-3rkghv.streamlitapp.com/

## Requirements
- easyocr==1.5.0
- numpy==1.22.4
- Pillow==9.1.1
- streamlit==1.10.0
- torch==1.11.0

## Installation

Must have to satisfy all the requirements

```bash
  streamlit run ocr.py
```

## FAQ

#### When to use it?
When you got somethings useful in paragraph/large text but don't want to write it manually then this tools for you.
